#  Weather Web App

A modern and responsive Weather Web Application built using **HTML, CSS, JavaScript, and the OpenWeather API**. The application allows users to search for any city and instantly view real-time weather information through a clean and intuitive interface.

---

##  Live Demo

🔗 https://Atchut016.github.io/Weather-App/

##  Features

- Search weather by city name
-  Real-time temperature display
-  Humidity information
-  Wind speed details
-  Dynamic weather icons
-  Error handling for invalid city names
-  Fully responsive design
-  Modern and attractive user interface

---

##  Technologies Used

- HTML
- CSS
- JavaScript 
- OpenWeather API

---



## How It Works

1. Enter the name of any city.
2. Click the search button.
3. The application sends a request to the OpenWeather API.
4. The API returns real-time weather data.
5. The application displays:
   - Temperature
   -  Humidity
   -  Wind Speed
   -  Weather Condition
   -  City Name
